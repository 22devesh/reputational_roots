## Recording Tips

1. **Prepare Environment:**
   - Clean desktop
   - Close unnecessary apps
   - Full screen recordings
   - Good lighting if showing face

2. **Recording Tools:**
   - OBS Studio (free)
   - Loom
   - QuickTime (Mac)
   - Screen recorder built into Windows

3. **Mobile Recording:**
   - Use screen recording on phone
   - Or use iOS Simulator/Android Emulator

4. **Editing:**
   - Add text overlays for key points
   - Smooth transitions
   - Background music (optional)
   - Export in HD (1080p)

5. **Upload:**
   - YouTube (Unlisted)
   - Google Drive
   - Vimeo
